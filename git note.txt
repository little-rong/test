git工作区域：
git repository(git 仓库)：最终确定的文件保存到仓库，成为一个新的版本，并且对他人可见
暂存区：暂存已经修改的文件最后统一提交到git仓库中
工作区（working directory）添加、编辑、修改文件等动作
